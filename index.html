After several years working in Loans Operations and trying my hand at teaching English, I turned to Web Development.

This appears to be the best professional decision I've made as I love the challenges and continuous learning that is part of this type of work. To see and be part of an evolving project is very rewarding.

My Projects

Pub Hop

A mobile first app designed for walking / sightseeing / pub enthusiasts using a MEAN stack with the Google Maps / Places API. Users can search for and create pub crawls/walks in London.

PetSitters

Group project using Node.js with AJAX which aimed to provide a service for pet owners who want their pets and house to be looked after whilst they are on holiday.

FX:Rigs

A resource for guitarists interested in guitar effects pedals and ‘rigs’ using Ruby on Rails. A user can post and view information on famous guitarists pedal setups and see info about the pedals.

The Rorschach Memory Test

A twist on the classic game of memory for two players using JavaScript. Players match to halves of the same image. Once matched the full image appears under their scoreboard.
